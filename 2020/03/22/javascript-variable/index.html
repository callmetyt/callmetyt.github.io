思考题2

问题是优化以下代码：

由题目中的代码可以看出程序运行的情况：
函数不断地调用自身，同时也在不断地开辟栈空间用来保存自身上下文环境，从而保存当前函数运行的数据状态，那么栈空间会被迅速占满，从而发生栈溢出，程序出现错误。

递归算法可以是代码更容易读懂，但递归总可以用循环代替，而循环不会创建函数上下文空间保存数据，在栈上占用一段空间，并不断地对上面的数据进行运算，就不会造成栈溢出。
