第七周-typescript高级类型学习2

前端 typescript
发布日期:   2020-08-23

字面量类型

字符串

  • 基础

    type Name = "tyt";
let test: Name = "tyt";
    • test的取值只能是tyt

  • 联合类型

    type Names = "tyt" | "lyx";
let future: Names = "lyx";
    • future只能是tyt或者lyx
    • 编译器会自动出现代码提示

数字

  • 效果和字符串字面量一样

可辨识联合（标签联合）

要素

  • 具有普通的单例类型属性
  • 一个类型别名包含了某些类型的联合

例子

interface Square {
  kind: "square";
  size: number;
}

interface Rectangle {
  kind: "rectangle";
  height: number;
  width: number;
}

interface Circle {
  kind: "circle";
  radius: number;
}

type Shape = Circle | Rectangle | Square;

function getArea(s: Shape) {
  switch (s.kind) {
    case "square":
      return s.size * s.size;
      break;
    case "rectangle":
      return s.height * s.width;
      break;
    case "circle":
      return s.radius ** 2 * Math.PI;
  }
}

  • 可辨识联合类型要素

    • 普通的单例类型

      interface Square {
  kind: "square";
}
interface Rectangle {
  kind: "rectangle";
}
interface Circle {
  kind: "circle";
}

      kind即是

    • 一个类型别名包含了某些类型的联合

      type Shape = Circle | Rectangle | Square;

      Shape类型别名包含了三个类型

  • 最后形成可辨识联合

    • Ts可以直接提供代码提示

完整性检查

  • 方法1:

    function getArea(s: Shape): number {
  switch (s.kind) {
    case "square":
      return s.size * s.size;
      break;
    case "rectangle":
      return s.height * s.width;
      break;
    // case "circle":
    //   return s.radius ** 2 * Math.PI;
  }
}
//报错
    • 因为有三个类型，一个没有判断，没有返回语句（默认返回undefined），返回值类型是numebr，则会报错

this类型

  • 和标准的JavaScript中的this一样

  • 使用tip：

    //链式调用的实现
class Counter {
  constructor(public count: number = 0) {}
  add(val: number) {
    this.count += val;
    return this;
  }
  subtract(val: number) {
    this.count -= val;
    return this;
  }
}

let counter = new Counter(9);
console.log(counter.add(3).subtract(9));
//Counter {count: 3}
    • 链式调用的根本就是每次调用方法后返回原来的对象（具有一样的方法）

索引类型

keyof

  • 索引类型查询操作符

    • 运算一个类型返回这个类型所有属性名组成的联合类型
    • 注意是所有属性名，不是值

  • 例子

    interface Info {
  name: string;
  age: number;
}

let infoTest: keyof Info;
infoTest = "name";
infoTest = "age";
infoTest = "haha";//报错
    • infoTest的值只能是接口Info里的name或者age，keyof取的是属性名（键）

索引访问操作符 []

  • 可以访问对象的属性（JavaScript也可）

    interface Type {
  a: never;
  b: number;
  c: string;
  d: boolean;
  e: object;
  f: Function;
  g: undefined;
  h: null;
  i: bigint;
  j: symbol;
}

type test = Type[keyof Type];
    • test支持除了never以外的Type接口定义的所有数据类型****
    • keyof不会返回never数据类型的属性名
    • []可以访问对象的属性值
